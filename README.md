Workflow工作流是一款管理任务流程的软件，类似于Activity工作流。Workflow的优势是解耦业务功能，只提供流程相关接口，功能灵活，易于扩展。
主要功能包括：“流程定义”，“流程设计”，“流程待办/已办”，“流程跟踪”等。
此项目为后端模块，只提供接口，前端页面请参考https://github.com/wanli420122/workflow_web.git
